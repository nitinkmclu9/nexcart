🚀 Deploy NexCart to Render

Step-by-Step Visual Guide - Takes 15 minutes

1

Create MongoDB Database (5 min)

Go to: MongoDB Atlas Sign Up

1. Sign up with Google (fastest)
2. Click "Build a Database" → Choose FREE (M0)
3. Create user: nexcartadmin / (your password - SAVE IT!)
4. Allow access from: 0.0.0.0/0 (Anywhere)
5. Click "Connect" → "Connect your application"
6. Copy connection string and replace <password> with your actual password
7. Add /nexcart at the end

mongodb+srv://nexcartadmin:YOUR_PASSWORD@cluster0.xxxxx.mongodb.net/nexcart?retryWrites=true&w=majority
2

Prepare Backend ZIP (1 min)

Run this script: Double-click prepare-for-render.bat in the nexcart folder

OR manually:
1. Go to: C:\Users\nitin\OneDrive\E comerse website\nexcart
2. Right-click backend folder
3. Click "Send to" → "Compressed (zipped) folder"
4. This creates backend.zip

3

Deploy to Render (7 min)

Go to: Render.com

1. Sign up with GitHub or Google
2. Click "New +" → "Web Service"
3. Click "Upload a ZIP file"
4. Upload your backend-for-render.zip

Configure Settings:

Setting Value
Name nexcart-backend
Region Oregon (US West)
Runtime Node
Build Command npm install && npm run build
Start Command node dist/server.js
Plan Free
4

Add Environment Variables ⚠️ IMPORTANT

Click "Advanced" → "Add Environment Variable" and add these 6 variables:

KEY VALUE
NODE_ENV production
PORT 10000
MONGODB_URI Your MongoDB connection string from Step 1
JWT_SECRET nexcart-super-secret-jwt-key-2024
JWT_EXPIRE 7d
FRONTEND_URL https://frontend-fawn-psi-99.vercel.app
⚠️ Important: Replace MONGODB_URI with your ACTUAL MongoDB connection string from Step 1!
5

Deploy & Test (2 min)

1. Click "Create Web Service"
2. Wait 3-5 minutes for deployment
3. When it shows "Live", test it:

https://nexcart-backend-xxxxx.onrender.com/api/health

You should see:

{ "status": "OK", "message": "NexCart API is running" }
6

Connect Frontend (2 min)

Go to: Vercel Dashboard

1. Click your project: nexcart-frontend
2. Go to: Settings → Environment Variables
3. Add new variable:
   - Name: NEXT_PUBLIC_API_URL
   - Value: https://nexcart-backend-xxxxx.onrender.com/api
   - Environment: ✅ Production
4. Click "Save"
5. Go to Deployments → Redeploy latest

🎉 Deployment Complete!

Your Live URLs:

Frontend: https://frontend-fawn-psi-99.vercel.app

Backend: https://nexcart-backend-xxxxx.onrender.com
